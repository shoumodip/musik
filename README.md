# smm
Simple Music Manager

## Dependencies
- [SDL2](https://www.libsdl.org/)
- [SDL2 Mixer](https://www.libsdl.org/projects/SDL_mixer/)

## Installation
```console
$ ./build.sh
$ ./build.sh <playlist>
```

## Playlists
Smm reads playlist information from `~/.config/smm.conf`.

```conf
# Playlist Name
<file-1>
<file-2>
<file-3>
...

# Another Playlist
<file-1>
<file-2>
...
```
